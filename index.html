Hjälp för iPhone och Mac

Vad behöver du hjälp med?

Välj ett ämne i menyn eller tryck på en vanlig fråga här nedanför.

Fästa ämnen

Vanliga frågor

Den här sidan är en enkel hjälpsida och är inte kopplad till Apple.

Wi-Fi fungerar inte

  1. Öppna Inställningar.
  2. Tryck på Wi-Fi.
  3. Kontrollera att Wi-Fi är påslaget.
  4. Välj ditt nätverk i listan.

Hur öppnar jag Anteckningar?

iPhone

  1. Gå till hemskärmen.
  2. Leta efter appen Anteckningar.
  3. Tryck på appen för att öppna den.

Mac

  1. Öppna Anteckningar på din Mac.
  2. Klicka på en anteckning i listan.
  3. Börja skriva.

Radera en anteckning

  1. Öppna Anteckningar.
  2. Svep åt vänster på anteckningen.
  3. Tryck på Radera.
  1. Öppna Anteckningar.
  2. Markera anteckningen du vill ta bort.
  3. Tryck på backsteg och bekräfta.

Skapa en mapp i Anteckningar

  1. Öppna Anteckningar.
  2. Tryck på Ny mapp.
  3. Ge mappen ett namn.
  4. Tryck på Spara.
  1. Öppna Anteckningar på Mac.
  2. Klicka på Ny mapp.
  3. Ge mappen ett namn.
  4. Klicka på Skapa.

Prata så att det blir text

  1. Öppna ett textfält i Anteckningar.
  2. Tryck på mikrofonen på tangentbordet.
  3. Börja prata tydligt.
  4. Tryck på mikrofonen igen när du är klar.
  1. Öppna Anteckningar på Mac.
  2. Klicka där du vill skriva.
  3. Välj Redigera i menyn högst upp.
  4. Klicka på Börja diktera.
  5. Börja prata.

Tips: Använd gärna hörlurar med mikrofon för bättre ljud.

Hur startar jag om enheten?

iPhone

  1. Håll in sidoknappen.
  2. Håll samtidigt in en volymknapp.
  3. Dra reglaget Stäng av.
  4. Starta igen med sidoknappen.

Mac

  1. Klicka på Apple-menyn.
  2. Välj Starta om.
  3. Klicka på Starta om igen.

Hur speglar jag till TV?

  1. Dra ner från övre högra hörnet på skärmen.
  2. Tryck på Skärmdubblering.
  3. Välj din TV i listan.

Kontrollcenter är menyn med snabba inställningar.

Hur justerar jag ljudet?

iPhone

  1. Tryck på volymknapparna på sidan.
  2. Öppna Kontrollcenter.
  3. Dra reglaget för ljud upp eller ner.

Mac

  1. Klicka på högtalarikonen i menyraden.
  2. Dra reglaget upp eller ner.
  3. Testa ljudet med valfri video eller musik.

Hur söker jag?

iPhone

  1. Svep ner på hemskärmen.
  2. Eller tryck på förstoringsglaset.
  3. Skriv det du letar efter.
  4. Tryck på appen eller resultatet.

Mac

  1. Tryck på kommandoknappen (⌘) och mellanslag.
  2. Skriv det du letar efter.
  3. Tryck på programmet eller dokumentet i listan.

Spotlight är Macens sökfunktion.

Hur tar jag en skärmbild?

iPhone

  1. Tryck på sidoknappen.
  2. Tryck samtidigt på volym upp.
  3. Släpp knapparna.
  4. Skärmbilden visas nere i hörnet.

Mac

  1. Tryck på kommandoknappen (⌘), shift och siffran 4 samtidigt.
  2. Muspekaren blir ett kors.
  3. Dra över området du vill ta bild på.
  4. Skärmbilden sparas på skrivbordet.

Hur skickar jag en skärmbild i ett meddelande?

  1. Ta en skärmbild.
  2. Tryck på bilden nere i hörnet.
  3. Tryck på dela-knappen.
  4. Välj Meddelanden.

Kontakter

Lägg till en kontakt

  1. Öppna appen Kontakter.
  2. Tryck på plus-knappen.
  3. Skriv namn och telefonnummer.
  4. Tryck på Klar.

Radera en kontakt

  1. Öppna appen Kontakter.
  2. Öppna kontakten du vill ta bort.
  3. Tryck på Redigera och sedan Radera kontakt.

Lägg till en kontakt

  1. Öppna appen Kontakter på Mac.
  2. Klicka på plus-knappen.
  3. Skriv namn och telefonnummer.
  4. Klicka på Klar.

Radera en kontakt

  1. Öppna appen Kontakter.
  2. Markera kontakten du vill ta bort.
  3. Tryck på backsteg och bekräfta.

Program

  1. Tryck på kommandoknappen (⌘) och mellanslag.
  2. Skriv namnet på programmet.
  3. Tryck på programmet i listan.

Du kan också hitta program genom att klicka på Appar i raden längst ner på skärmen.

Hur raderar jag gamla iMessage?

  1. Öppna Meddelanden.
  2. Svep åt vänster på ett meddelande.
  3. Tryck på Radera.
